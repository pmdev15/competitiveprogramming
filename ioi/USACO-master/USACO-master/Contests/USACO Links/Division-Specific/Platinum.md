# Key

 * Graphs
	* SCC = Strongly Connected Components
	* BCC = Biconnected Components
	* LCA = Lowest Common Ancestor
 * SegTree = Segment Tree
 * D&C = Divide & Conquer
 * BigNums = Big Numbers
 * Sqrt = Square Root
 * SA = Suffix Array
 * BBST = Balanced Binary Search Tree (ex. Treap)
